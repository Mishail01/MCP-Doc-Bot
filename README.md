# MCP Developer Documentation Bot

## Overview

This bot automatically generates developer documentation from Python codebases.
It can:

- Scan a repo and extract functions, classes, and docstrings
- Generate README.md and API docs
- Explain functions or classes interactively via MCP (mocked locally)

## How to Run

1. Open PowerShell in the project folder
2. Install dependencies and generate docs:

   ```powershell
   .\run_bot.ps1
   ```
